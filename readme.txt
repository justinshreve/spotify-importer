Automatically publish songs from a Spotify Playlist.

== Description ==

This plugin depends on having both https://wordpress.org/plugins/keyring/ and https://wordpress.org/plugins/keyring-social-importers/ installed.

This plugin gives you the ability to pull in Spotify songs from a specific playlist and re-publish it on your own WordPress site.

After an initial import, you can also optionally check each hour and automatically download new content.

== Installation ==

0. Install and activate the [Keyring plugin](http://wordpress.org/extend/plugins/keyring/), which is required for authentication
1. Install and activate the [Keyring social importers plugin](http://wordpress.org/extend/plugins/keyring-social-importers/)
2. Install and activate Spotify Importer either via the WordPress.org plugin directory, or by uploading the files to your server
4. Go to Tools > Import > (service) and follow the prompts
